# And Chess For All Website

A responsive static website designed for GitHub Pages.

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload every file and the `assets` folder from this package.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**, choose `main` and `/root`, then save.

## Included features
- Time-of-day greeting using the visitor’s browser clock.
- Chess.com public API dashboard with graceful fallback.
- Club registration page linked to Tally.
- Recommended course affiliate page.
- Google Maps embed for local chess locations.
- Quiet blue/red knight patrol animation with no audio and an on/off control.
- Reduced-motion accessibility support.
